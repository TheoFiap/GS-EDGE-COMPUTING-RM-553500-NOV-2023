RM-553500-GS-NOV-2023

O PROBLEMA:
Uma noite mal dormida pode agravar a condição de alguém doente ou em tratamento devido a diversos fatores. A falta de sono adequado compromete o sistema imunológico, dificulta a regeneração celular, aumenta os níveis de estresse, desregula a produção hormonal e prejudica a capacidade de concentração e coordenação. Esses efeitos podem interferir negativamente no processo de recuperação, tornando essencial um sono adequado para a melhoria da saúde.

A SOLUÇÃO:
Este código Arduino controla a abertura e fechamento automáticos de uma janela com base na luminosidade ambiente. Um sensor de luminosidade (LDR) monitora a claridade, e a janela abre quando detecta luz suficiente durante o dia e fecha quando escurece à noite. O código também permite o controle manual da janela via comunicação serial. Os tempos de abertura e fechamento devem ser configurados conforme cada janela/cortina

Para configurar, basta cronometrar quanto tempo leva para sua janela / cortina abrir e quanto para fechar. Coloque esses valores em
int tempoFecha
int tempoAbre
Assim o programa acina de acordo com o local onde o sistema for instalado!
